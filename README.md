# JunctionX Seoul 2020 | Weco | Team Telepa-see


### Description

The implementation of frontend is not fully complete and does not have proper binding with backend. However, it still demonstrates basic understanding of our product.
Frontend of Weco is created using Javascript, Express, Node.js, Socket.io to connect to the backend.
